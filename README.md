# Operation: Paradox Tiles

## Overview
_Operation: Paradox Tiles_ is a single-player first-person puzzle platformer set inside a malfunctioning simulation chamber. The player must navigate a grid of reactive tiles, each altering physics, and reach the exit before the time runs out.

## Core Features
* Tile-based trigger system
* Dynamic physics (e.g. gravity shifts and teleportation)
* Changing controls
* Time-based escape challenge
* Progressive difficulty

## Gameplay
You play as a spy trapped in an unstable simulation. Each tile you step on triggers an effect, either positive or negative. Leaarn tile behaviours, adapt and find the exit before the timer expires.

## Controls
* Arrow keys: movement
* Space: jump

## Tile Mechanics
* Gravity inversion
* Time warp
* Slow field
* Teleport
* Phantom exit

## Objective
Reach the exit door before the timer reaches zero.

## Win/Lose Condition
* **Win**: reach the exit in time
* **Lose**: timer runs out
